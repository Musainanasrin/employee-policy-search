# Employee Policy Search System Using Azure AI Search

## Project Overview
The **Employee Policy Search System** is a cloud-based application developed using **Azure AI Search** to help employees quickly retrieve HR and company policy documents. The system provides an intelligent, keyword-based search mechanism that improves accessibility and efficiency.

This project was developed as part of an **Industry Internship Program at Tata Consultancy Services (TCS)*

##  Objectives
- Provide fast access to employee policy documents  
- Reduce dependency on HR for policy-related queries  
- Implement a cloud-based intelligent search system  
- Demonstrate practical usage of Azure AI Search  

---

## Problem Statement
Employees face difficulty in quickly accessing accurate policy information due to the lack of a centralized intelligent search system.

---

## System Architecture
- **Azure Blob Storage** – Stores employee policy documents  
- **Azure AI Search** – Indexes and retrieves documents  
- **Azure Static Web Apps** – Hosts the web application  
- **REST API** – Fetches search results from Azure AI Search  

---

##  Implementation Summary
1. Uploaded policy documents to Azure Blob Storage

3. Configured searchable and filterable fields  
4. Built a web interface to query the search index  
5. Deployed the application using Azure Static Web Apps
